# مستندات پروژه NewEarthCoop

این مستندات پروژه NewEarthCoop را پوشش می‌دهد، یک پلتفرم همکاری اجتماعی-اقتصادی مبتنی بر Laravel با ویژگی‌های پیشرفته مانند گروه‌بندی هوشمند کاربران، چت بلادرنگ، مزایده‌ها و پشتیبانی از هوش مصنوعی نجم هودا.
