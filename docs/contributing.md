# راهنمای مشارکت

از علاقه شما به مشارکت در پروژه NewEarthCoop سپاسگزاریم!

## نحوه مشارکت
1. مخزن را fork کنید.
2. یک شاخه جدید برای ویژگی یا رفع باگ ایجاد کنید.
3. تغییرات را اعمال کنید و تست کنید.
4. یک Pull Request ارسال کنید با توضیح تغییرات.

## استانداردها
- کد را با PSR-12 فرمت کنید.
- کامنت‌های معنی‌دار اضافه کنید.
- تست‌ها را بنویسید.
- مستندات را به‌روزرسانی کنید.

## گزارش باگ‌ها
باگ‌ها را در بخش Issues گزارش کنید با جزئیات کامل.

## تماس
برای سؤالات، ایمیل support@newearthcoop.com را استفاده کنید.
