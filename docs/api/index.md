# مستندات API

این بخش مستندات API برنامه NewEarthCoop را پوشش می‌دهد.

## مقدمه
API بر اساس RESTful طراحی شده و از Laravel Sanctum برای احراز هویت استفاده می‌کند.

## نقاط پایانی اصلی
- `/api/user`: اطلاعات کاربر
- `/api/groups`: مدیریت گروه‌ها
- `/api/chat`: پیام‌ها
- `/api/auctions`: مزایده‌ها

## احراز هویت
برای دسترسی به API، توکن Bearer ارسال کنید.

## مثال‌ها
```
GET /api/user
Authorization: Bearer {token}
```

*(این بخش در حال توسعه است. برای جزئیات کامل، کد منبع را بررسی کنید.)*
