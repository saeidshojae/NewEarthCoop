# راهنمای استقرار

این راهنما نحوه استقرار برنامه NewEarthCoop در محیط تولید را توضیح می‌دهد.

## پیش‌نیازها
- سرور با PHP 8.1+
- MySQL یا PostgreSQL
- Composer
- Node.js و NPM
- وب‌سرور (Apache/Nginx)

## مراحل استقرار
1. کد را از مخزن Git clone کنید.
2. وابستگی‌ها را نصب کنید: `composer install --no-dev`
3. فایل `.env` را پیکربندی کنید با تنظیمات پایگاه داده و سایر متغیرها.
4. کلید برنامه را تولید کنید: `php artisan key:generate`
5. مهاجرت‌ها را اجرا کنید: `php artisan migrate --seed`
6. دارایی‌ها را کامپایل کنید: `npm run build`
7. مجوزها را تنظیم کنید: `chmod -R 755 storage bootstrap/cache`
8. وب‌سرور را پیکربندی کنید تا به `public/` اشاره کند.
9. صف‌ها را راه‌اندازی کنید: `php artisan queue:work`
10. کش را پاک کنید: `php artisan config:cache`

## ملاحظات امنیتی
- از HTTPS استفاده کنید.
- متغیرهای محیطی را ایمن نگه دارید.
- به‌روزرسانی‌های امنیتی را اعمال کنید.

## مانیتورینگ
- لاگ‌ها را در `storage/logs/` بررسی کنید.
- از ابزارهای مانیتورینگ مانند Laravel Telescope استفاده کنید.
